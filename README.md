# Blight Detector

Real-time Sol's RNG biome monitor with Discord webhook notifications.

## Usage

1. Download the latest release
2. Launch Sol's RNG in Roblox
3. Run `blight.exe`
4. Configure your Discord webhook URL in Settings
5. Press **START**

## Files

- `blight.exe` — desktop UI
- `detector.exe` — detection engine
- `flutter_windows.dll` — Flutter runtime
- `data/` — app assets
- `config.example.json` — rename to `config.json` and edit
